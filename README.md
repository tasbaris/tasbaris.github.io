# Barış Taş - Developer Portfolio

A minimalist, high-performance developer portfolio built with semantic HTML5, modern CSS, and vanilla JavaScript. Designed to showcase backend engineering expertise through a clean, serious, and zero-distraction interface.

## 🎯 Architectural Philosophy

This project was built from scratch rejecting heavy frontend frameworks in favor of absolute control over the DOM and CSS Object Model.

- **Zero Dependencies:** No React, Vue, or Tailwind. Only raw Web APIs.
- **Monochromatic Aesthetic:** Focused entirely on content and typography (Inter font).
- **Performance First:** No layout shifts, optimized paint cycles, and hardware-accelerated animations.
- **Dynamic Integration:** Asynchronous fetching of open-source projects directly via the GitHub REST API.

## 🚀 Features

- **Responsive Grid Layout:** CSS Grid architecture adapting seamlessly from 4K displays to mobile screens.
- **Intersection Observer Animations:** Content gracefully reveals as it enters the viewport, minimizing initial load payload.
- **Automated Repository Sync:** The `Projects` section automatically syncs with GitHub, filtering out forks to display only original backend tooling and APIs.
- **Semantic Structure:** Fully accessible markup ensuring screen-reader compatibility and SEO optimization.

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** CSS3 (Custom Properties, Grid, Flexbox, Keyframes)
- **Logic:** Vanilla JavaScript (ES6+, Async/Await, IntersectionObserver API)
- **Data Source:** GitHub REST API v3
