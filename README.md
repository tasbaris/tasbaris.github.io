## 🌍 English

Welcome to my high-performance, interactive portfolio ecosystem. This isn't just a static site; it's a **Vanilla JS Engine** showcase featuring a custom i18n system, real-time API integrations, and two fully developed browser games.

### 🚀 Key Engineering Features

- **⚡ Performance First:** Zero dependencies. No heavy frameworks (React/Vue/Angular). Just optimized Vanilla JS and CSS for near-instant Load Times and 100/100 Lighthouse scores.
- **🌐 Custom i18n Engine:** A proprietary localization system built from scratch. It supports instant EN/TR toggling with state persistence.
- **📱 True Cross-Platform:** Responsive from 320px to 4K. Includes a custom **Virtual Keyboard API** for interactive games on mobile devices.
- **📡 Live Data Sync:** Real-time integration with the GitHub REST API to showcase active repositories.
- **🎨 Dynamic Theming:** Intelligent Light/Dark mode switcher with OS-level preference detection.

## 🇹🇷 Türkçe

Yüksek performanslı, interaktif portfolyo ekosistemime hoş geldiniz. Bu sadece statik bir web sitesi değil; özel bir i18n sistemi, gerçek zamanlı API entegrasyonları ve iki adet tam teşekküllü tarayıcı oyununu barındıran bir **Vanilla JS** gövde gösterisidir.

### 🚀 Öne Çıkan Mühendislik Özellikleri

- **⚡ Önce Performans:** Sıfır bağımlılık. Framework (React/Vue) yok. Anlık yükleme süreleri ve 100/100 Lighthouse skorları için optimize edilmiş Saf JS ve CSS.
- **🌐 Özel i18n Motoru:** Sıfırdan inşa edilmiş yerelleştirme sistemi. Oturumlar arası kalıcı durum desteği ile anlık EN/TR geçişi.
- **📱 Tam Mobil Uyumluluk:** 320px'den 4K'ya kadar kusursuz görünüm. Mobil cihazlarda oyun oynanabilirliği için geliştirilmiş **Sanal Klavye API**'si.
- **📡 Canlı Veri Senkronizasyonu:** GitHub REST API ile gerçek zamanlı entegrasyon sayesinde en güncel projelerin otomatik sergilenmesi.
- **🎨 Dinamik Temalandırma:** İşletim sistemi tercihlerini algılayan akıllı Açık/Koyu (Light/Dark) tema anahtarı.
